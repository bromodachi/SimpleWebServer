Welcome to the simple webserver.

This project is just to play around with how web servers work. For production usage, we recommend spring.

We only handle GET requests for simplicity sakes. To handle POST requests, it requires parsing the JSON byte/string and not sure if it's really worth doing on a project that's supposed to be simple.

Here are some of the requests you can do for GET!

  • "/" → shows this page.
  • "/index.html" → brings you to another hello world page.
  • "/hello-world" → Gives you a response in text/plain that simply says hello world.
  • "/nested-json" → Gives you a response in application/json.
Everything else, you will get a response that simply states "The content you're looking for is not found".
